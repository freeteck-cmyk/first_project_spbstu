# Поиск самой длинной подстроки в списке объектов
Программа анализирует переданный список, находит в нем самую длинную строку и извлекает из нее первую встретившуюся подстроку максимальной длины без повторяющихся символов.
# Функционал
Программа разделена на три основные функции:
1. `check_input(element)`: Проверяет, является ли элемент списка строкой. Возвращает `True` или `False`.
2. `get_substring(s)`: Находит первую самую длинную подстроку без повторяющихся символов в переданной строке.
3. `main(data_list)`:
* Принимает список объектов.
* Находит индекс самой длинной строки (если строк несколько одинаковой длины — выбирает первую).
* Если строк нет, возвращает `-1` и пустую строку.
* Выводит в консоль индекс строки и результат работы `get_substring`.
# Пример работы
## Входные данные:
`[[0, 4, 8], 'abca', 7, 'aaaaa']`
## Логика:
Самая длинная строка — `'aaaaa'` (индекс 3).
В строке `'aaaaa'` самая длинная подстрока без повторений — `'a'`.
## Выходные данные:
`3, 'a'`
